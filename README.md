# 💍 Kasun & Himasha — Wedding Invitation

A luxury Sri Lankan traditional Kandyan-style wedding invitation website.

**Live Site:** https://[your-username].github.io/kasun-himasha-wedding/

## ✨ Features
- Sinhala & English bilingual content
- Animated golden particles & floating lotus petals
- Kandyan-inspired decorative frames & ornaments
- Live countdown timer to 10 September 2026
- Temple bell audio (Web Audio API)
- Fully responsive — mobile, tablet, desktop
- Framer Motion animations throughout

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repo named `kasun-himasha-wedding`
2. Push this code to the `main` branch
3. Go to **Settings → Pages → Source → GitHub Actions**
4. The site auto-deploys on every push!

## 🛠 Local Development

```bash
npm install
npm run dev
```

## 🏗 Build
```bash
npm run build
```

Built with React + Vite + Tailwind CSS + Framer Motion
